# hypedexer-sdk

TypeScript SDK for the Hypedexer Hyperliquid indexer API.

> **Status:** pre-release (`v0.0.0`). Public API not stable yet.

## Packages

- [`@hypedexer/sdk`](./packages/sdk) — the SDK itself.

## Development

```bash
pnpm install
pnpm typecheck
pnpm test
pnpm build
```

`pnpm check` runs lint + typecheck + test + build in one shot.
